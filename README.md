## Installation

1. Using `Lazy`:

```lua
{ 'datsfilipe/min-theme.nvim' }
```

2. Using `Packer`:

```lua
use 'datsfilipe/min-theme.nvim'
```

## Configuration

For now it's in early stage, so if you want to use just do `:colorscheme min-theme`.
